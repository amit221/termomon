System Summary

Compi is a terminal creature collection game where creatures have 4 trait slots (eyes, mouth, body, tail), each progressing through 8 rarity tiers from Common to Eternal. Progression is driven by a merge system where sacrificing "food" creatures upgrades a target creature's slots. Each slot requires 35 merge steps (28 star upgrades + 7 cross-rarity jumps) for a total of 140 steps to fully max one creature. The harmonic signature system provides strategic depth: every trait has a mathematical signature, and choosing food creatures with optimal resonance yields up to +20% merge bonus. A pity timer guarantees progress even with bad luck, keeping the P95/P50 variance ratio at an excellent 1.07. Strategic players complete in ~44 days, random players in ~51 days -- a meaningful but non-punishing 13.6% advantage for informed play.

Pity Timer System

How it works: After threshold / 2 consecutive failures, a linear bonus ramps up. At threshold failures, success is guaranteed (100%).


Example: Legendary jump (threshold = 4)

Attempt 152% base
Attempt 252% base
Attempt 352% + ~24% pity = 76%
Attempt 4Guaranteed 100%

This keeps the P95/P50 variance ratio at 1.07 -- bad luck barely extends the journey.
