---
name: list
description: Show all available Termomon slash commands
---

Display the following list of available Termomon commands to the user:

| Command | Description |
|---------|-------------|
| `/tm:scan` | Show nearby creatures that can be caught |
| `/tm:catch` | Attempt to catch a nearby creature |
| `/tm:collection` | Browse your caught creatures and evolution progress |
| `/tm:evolve` | Evolve a creature that has enough fragments |
| `/tm:inventory` | View your items |
| `/tm:status` | View your player profile and game stats |
| `/tm:settings` | View or change game settings |
| `/tm:list` | Show this list of commands |
